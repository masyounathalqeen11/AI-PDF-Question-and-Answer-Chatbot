AI PDF Question Answer Chatbot

This project is a simple AI-powered chatbot that answers questions from uploaded PDF documents.

Feature:
1.Upload PDF
2.Extract text
3.Ask questions
4.AI-generated answers
5.Technologies
6.Python
7.Google Colab
8.LangChain
9.Gemini/OpenAI 
10.PyPDF
11.FAISS 

Future Improvements:
Streamlit frontend, Multiple PDF support, Chat history, Deployment
